## 2026-06-16 — v0.1.0

### Summary
Initial public release of Freelancer Web Toolbox as a dependency-free static browser application.

### Changed Files
- `public/index.html`
- `public/assets/css/app.css`
- `public/assets/js/*`
- `docs/*`
- `ai/*`
- `examples/*`
- root documentation and metadata files

### Added
- EN/RU/KK multilingual interface with LocalStorage language persistence.
- Seventeen offline tools for development, marketing, CRM, SEO, text, business, and security workflows.
- Saved results panel using local browser storage.
- Validation script using only Node.js built-in modules.
- Documentation, examples, license, security policy, and contribution guide.

### Fixed
- Not applicable for the initial release.

### Security
- No backend, database, analytics, CDN, external API keys, or tracking scripts.
- User-entered data is displayed through text-safe controls or `textContent`.
- LocalStorage privacy warning is included in the app and documentation.

### Notes
- Future releases can add export/import for saved results and optional PWA enhancements.
