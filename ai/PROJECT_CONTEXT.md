Freelancer Web Toolbox is a public static GitHub project. AI agents should treat it as a practical open-source toolbox, not a landing page or toy demo.

The key architectural promise is simple: open `public/index.html` and use the tools without a backend, database, build step, external CDN, API key, account, or network request.

The app supports English, Russian, and Kazakh. All user-facing strings for the main UI and tools should be translated through `translations.js`.

Privacy matters more than feature bloat. Tools should help with local text and link work, but should not encourage users to paste secrets or private customer data.
