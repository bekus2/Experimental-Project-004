## Dashboard

- Search tools by name, description, or category.
- Filter by Developer, Marketing, CRM, Business, SEO, Text, and Security.
- Open tools in a focused workspace.

## Tools

- JSON Formatter and Validator: validate, format, and minify JSON.
- Base64 Encoder / Decoder: convert UTF-8 text to and from Base64.
- URL Encoder / Decoder: encode and decode URL components.
- UTM Link Builder: build campaign URLs with UTM parameters.
- WhatsApp Link Generator: create `wa.me` links from sanitized phone input.
- Telegram Link Generator: create `t.me` links from normalized usernames.
- Password Generator: generate local browser passwords.
- Slug Generator: transliterate Russian, Kazakh, and English text to Latin slugs.
- Meta Preview: preview title, URL, and description text.
- Robots.txt Generator: build starter crawler rules.
- Sitemap.xml Generator: build a simple sitemap XML document.
- Webhook Payload Viewer: format JSON and detect common fields.
- KZT Commission Calculator: estimate fees, tax, bonus, and net amount.
- Freelance Price Estimator: estimate project pricing from rate and hours.
- Invoice Text Generator: draft payment request text in EN/RU/KK.
- HTML Minifier: simple HTML whitespace helper.
- CSS Minifier: simple CSS whitespace helper.

## Saved Results

Generated outputs can be saved locally in the user's browser. Saved items can be copied, deleted individually, or cleared together.
