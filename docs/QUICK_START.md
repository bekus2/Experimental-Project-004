## Clone

```bash
git clone https://github.com/bekus2/Experimental-Project-004.git
cd Experimental-Project-004
```

## Open Directly

Open this file in a browser:

```text
public/index.html
```

## Run Local Server

```bash
python -m http.server 8000 -d public
```

Then open:

```text
http://localhost:8000
```

## Use Tools

1. Select English, Русский, or Қазақша.
2. Search for a tool or use a category.
3. Open the tool card.
4. Enter data.
5. Generate, copy, clear, or save the result.

## Deploy to GitHub Pages

Publish the repository from `main`. The root `index.html` redirects to `public/index.html`, so repository-root publishing is supported.
