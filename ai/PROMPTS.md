## Add a New Tool

```text
Add a new offline tool to Freelancer Web Toolbox. Preserve file:// support, no dependencies, EN/RU/KK translations, safe rendering, copy output, saved result support where useful, and update documentation.
```

## Improve UI

```text
Review the app UI for desktop and mobile. Improve spacing, readability, focus states, responsive behavior, and accessibility without changing the dependency-free architecture.
```

## Check Security

```text
Review user input handling, LocalStorage usage, and rendering. Fix unsafe output paths, avoid raw user input in innerHTML, and update SECURITY.md if needed.
```

## Add Translations

```text
Add or improve English, Russian, and Kazakh translations in translations.js. Keep keys aligned across en, ru, and kk.
```

## Refactor Tool Modules

```text
Refactor one or more tool modules for clarity while preserving behavior, file:// support, copy actions, saved results, and validation messages.
```

## Prepare GitHub Pages Deployment

```text
Check that root index.html redirects correctly, relative asset paths work, public/index.html runs through a static server, and deployment docs are accurate.
```

## Write Documentation

```text
Update README.md and docs/ to reflect the actual current behavior. Keep README bilingual in one file and avoid unsupported claims.
```

## Review Before Release

```text
Review the whole repository. Fix missing files, broken links, JavaScript errors, translation gaps, mobile layout issues, unsafe rendering, README inconsistencies, and validation script problems. Then run scripts/validate.mjs and summarize remaining limitations honestly.
```
